    PHP的函数中意想不到的坑

    2020-10-03
    develop, php
    php

    strtotime1

    1
2

    <?php
echo date("Y-m-d", strtotime("-1 month"));

    假设今天是2019-07-31, 输出： 2019-07-01

    1. 先做-1 month, 那么当前是07-31, 减去一以后就是06-31.
    2. 再做日期规范化, 因为6月没有31号, 所以就好像2点60等于3点一样, 6月31就等于了7月1

    Hugo KaTex 公式

    KaTeX简码可让您在markdown文档中呈现数学排版。见 KaTeX

    Example

    {{< katex [display] [class="text-center"]  >}}
x = \begin{cases}
   a &\text{if } b \\
   c &\text{if } d
\end{cases}
{{< /katex >}}
    \( x = \begin{cases} a &\text{if } b \\ c &\text{if } d \end{cases} \)

    Display Mode Example

    Here is some inline example: \(\pi(x)\) , rendered in the same line. And below is display example, having display: block \[ x = \begin{cases} a &\text{if } b \\ c &\text{if } d \end{cases} \] Text continues here.

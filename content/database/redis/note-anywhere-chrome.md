在网页右键菜单新增 新增note， 在网页任意位置点击新增，出现弹框，可以输入与网页相关的备注，  
可以调整文本框大小，可以调整背景颜色，可以固定pin在窗口某个位置；  

<!--more-->

<img src="../note-anywhere.jpeg" style="height:350px;"/>


当进入某个页面，根据 url 获取当前 url 是否有备注，存在则获取并在对应位置显示;

数据可同步到远程服务器；  
可以设置登录；




## 功能

- [ ] 长按添加
- [ ] 添加长按效果，可优化loading图
- [ ] 可拖拽
- [ ] 可缩放
- [ ] 可固定
- [ ] 可调配色
- [ ] 可搜索
- [ ] 可删除
- [ ] 存云端
- [ ] 从云端加载
- [ ] 可复制，不处理

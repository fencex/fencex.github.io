按钮是样式链接，可以链接到本地​​页面或外部链接。


## Example

```tpl
{{</* button relref="/" [class="..."] */>}}回到首页{{</* /button */>}}
{{</* button href="/posts" */>}}个人随笔{{</* /button */>}}
```

{{< button relref="/">}}回到首页{{< /button >}}
{{< button href="/posts" >}}个人随笔{{< /button >}}
